# 🤖 Telegram Bot Framework on Go

Простой и расширяемый каркас для создания Telegram ботов на языке Go с использованием библиотеки `telebot` v3.

## 📋 Особенности

- ✅ Готовая структура проекта
- ✅ Обработчики команд (/start, /help, /info)
- ✅ Обработка текстовых сообщений
- ✅ Инлайн-клавиатуры и callback-кнопки
- ✅ Middleware для логирования
- ✅ Поддержка состояний пользователей
- ✅ Загрузка конфигурации из .env
- ✅ Чистая архитектура с разделением логики
